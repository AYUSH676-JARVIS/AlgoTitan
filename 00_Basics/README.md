Basics - DSA Foundation
 Overview
This folder contains basic programming problems that build the foundation for Data Structures and Algorithms (DSA).
Before solving advanced problems, it is important to have a strong grip on:
	•	Logic building
	•	Problem-solving approach
	•	Code implementation
Objective
	•	Strengthen core programming concepts
	•	Improve logical thinking
	•	Prepare for DSA and coding interviews
	•	Build consistency in problem solving
Topics Covered
	•	Variables & Data Types
	•	Input / Output
	•	Conditional Statements (if-else)
	•	Loops (for, while)
	•	Functions
	•	Basic Mathematics
Why This Folder Matters
These problems are the building blocks for advanced topics like:
	•	Arrays
	•	Hashing
	•	Recursion
	•	Dynamic Programming
Strong basics = Faster problem solving + Better interview performance
Learning Outcome
After completing this folder, you will be able to:
	•	Write clean and logical code
	•	Understand problem patterns
	•	Transition smoothly to DSA topics
